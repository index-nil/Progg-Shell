┌---------------------------------------------------------┐
| Progg Shell - Shell on C with **not Standard** commands |
└---------------------------------------------------------┘

┌-----------------------------------Commands-----------------------------------┐
|                                                                              |
| ver    - Print Shell version                                                 |
| hello  - Print "Hello world!"                                                |
| quit , q , exit - Exit from Shell                                            |
| fl,dir,ls - Show objects in current folder                                   |
| del <Path + Full File name> - Delete file                                    |
| delay <ms> - Wait some time                                                  |
| prt <word> - Print string                                                    |
| prtread <Path + Full File name> - Print file content                         |
| noflag - Disable flags                                                       |
| exflag - Enable flags                                                        |
| run <command> - Send command to Windows                                      |
| crt <file name> - Creating or cleaning file                                  |
| cd  <folder name or path to folder> - Enter in the folder                    |
| cop, cln <Path_To_File> <Path_To_Duplicate> - Create Duplicate of file       |
| cmd    - Open System Command Prompt                                          |
| clv    - Clear view                                                          |
| cls    - Clear Screen                                                        |
| cop, cln <Path_To_File> <Path_To_Duplicate> - Create Duplicate of file       |
| create <object_name> - create object (For extensions developers)             |
| cpu-load - Show CPU load                                                     |
| mem-load - Show RAM load                                                     |
| mov <new_Path> - Move file                                                   |
| exec <Path + Full File name> - Execute file                                  |
| set-lng <language file name> - Sets language file to file in program folder  |
| mdr <folder name> - create folder in current path                            |
| ren <new_file_name> - Rename file                                            |
| neo <file_name or new_file_name> - Edit file with Neo                        |
|                                                                              |
└------------------------------------------------------------------------------┘
